## 🧪 Key Exercises

### 1. Fuzzy Set Representation ($\alpha$-cuts)
- [cite_start]Verified the representation of fuzzy sets using the Resolution Identity Principle[cite: 8, 26].
- [cite_start]Mapped membership values $\mu_A(x)$ across various levels ($L(A) = \{0.2, 0.4, 0.6, 0.8, 1\}$) to confirm set reconstruction[cite: 10, 15, 25].

### 2. Fuzzy Relation Composition
- [cite_start]Computed the composition of fuzzy relations using different operators to model system interactions[cite: 31].
- [cite_start]**Max-Min Composition:** Used for conservative estimation in relational modeling[cite: 33, 34].
- [cite_start]**Max-Product Composition:** Used for probabilistic-style relational weightings[cite: 35, 36].

### 3. Fuzzy Inference & Rule Evaluation
- [cite_start]Evaluated fuzzy rule outputs using **Minimum Implication** (Mamdani-style clipping)[cite: 41, 43].
- [cite_start]Modeled how input membership values $\mu_A(x')$ affect the resulting output membership functions $\mu_B(y)$[cite: 43, 47, 48].
